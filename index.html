About
›

BlackRoad builds sovereign compute infrastructure that runs on hardware you own. Five Raspberry Pi 5 boards, two Hailo-8 accelerators (52 TOPS), two cloud droplets, one WireGuard mesh.

No vendor lock-in. No SaaS dependencies. Everything self-hosted — Git (Gitea), AI inference (Ollama on 4 nodes), DNS (PowerDNS), object storage (MinIO), TLS (Caddy + Let's Encrypt), chat (RoundTrip), CI/CD (Gitea Actions), database (PostgreSQL on 3 nodes), cache (Redis).

The only external dependencies are Stripe (payment processing) and GoDaddy (domain registrar). Everything else runs on owned hardware or rented compute with full root access.
